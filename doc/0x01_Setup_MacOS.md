## **1. Установка Git**  

### **MacOS**  
1. **Способ 1** (через Homebrew):  
   ```bash
   brew install git
   ```  
2. **Способ 2** (через официальный установщик):  
   - Скачайте `.dmg` файл с [сайта Git](https://git-scm.com/download/mac).  
   - Откройте его и следуйте инструкциям.  

Проверка:  
```bash
git --version
```
