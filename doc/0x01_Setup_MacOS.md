## **1. Установка Git**  

### **MacOS**  
   **Способ 1** (через Homebrew):
   
   ```bash
   brew install git
   ```  

   **Способ 2** (через официальный установщик):  
   - Скачайте `.dmg` файл с [сайта Git](https://git-scm.com/download/mac).  
   - Откройте его и следуйте инструкциям.  

Проверьте установку:

```bash
git --version
```
